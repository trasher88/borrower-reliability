# Исследуем влияние семейного положения и количество детей на погашение кредитов

## Стек:
Python, Pandas, Matplotlib/Seaborn, Pymystem3
![Python](https://img.shields.io/badge/Python-3.9-blue?logo=python&logoColor=white)

## Структура проекта:
1. Загрузка и предварительный анализ данных
2. Предобработка данных
	- Обработка пропусков
	- Исправление аномалий
	- Обработка дубликатов
	- Лемматизация целей кредита
	- Категоризация данных
3. Анализ данных
	- Зависимость между количеством детей и возвратом кредита
	- Зависимость между семейным положением и возвратом кредита
	- Зависимость между уровнем дохода и возвратом кредита
	- Влияние целей кредита на возврат
4. Выводы

## Методы и подходы:
	- Предобработка данных: обработка пропусков, аномалий, удаление дубликатов, преобразование типов данных
	- Текстовая обработка (NLP): лемматизация, категоризация текста
	- Анализ данных: сводные таблицы, расчет целевых метрик, сравнительный анализ групп
	- Визуализация: столбчатые диаграммы, настройка стилей, отображение распределений и сравнение долей дефолтов
	- Работа с данными: Pandas(фильтрация, группировка, агрегация), квантильное ранжирование

## В ходе проекта решил следующие задачи:
	- Очистил и предобработал данные
	- Лемматизированы цели кредита и разделены на категории
	- Категоризированы: образование, семейное положение, тип и уровень дохода
	- Исследованы: связи между количеством детей и возвратом кредита, семейным положением и возвратом кредита, уровнем дохода и возвратом кредита, влияние целей кредита на возврат
	- Рассчитал процент дефолтов и построил наглядные визуализации
	- Выявил группы риска среди заемщиков
	- Сформулировал практические рекомендации для кредитного отдела банка
